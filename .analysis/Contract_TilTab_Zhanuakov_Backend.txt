В рамках реализации проекта «TilTap: AI for Multilingual Dialogue Without Borders» Фонд развития медиаконсалтинга в ЦА, в лице директора Токтогуловой Эльмиры Амановны, действующей на основании Устава, именуемая в дальнейшем «Заказчик», и Жанат Жанузаков, именуемый в дальнейшем «Исполнитель», заключили настоящий Договор о нижеследующем:
1. Предмет Договора
1.1. Исполнитель обязуется выполнить работы по проектированию и разработке серверной части приложения, REST API и интеграции AI-сервисов перевода в проекте TilTap, а Заказчик обязуется принять результаты работ и оплатить их в соответствии с условиями Договора.
1.2. В рамках настоящего Договора Исполнитель выполняет следующие функции: проектирование и разработка серверной части приложения на Node.js/TypeScript; настройка базы данных PostgreSQL; разработка REST API; реализация слоя перевода и AI-интеграций; обеспечение безопасности; оптимизация производительности; документирование кода (Swagger/OpenAPI).
1.3. Работы выполняются в соответствии с Техническим заданием (Приложение №1), являющимся неотъемлемой частью настоящего Договора.
2. Сроки выполнения работ
Работы выполняются поэтапно в период с 1 июня по 1 сентября 2026 года согласно плану проекта (Приложение № 2).
3. Права и обязанности сторон
Исполнитель выполняет работу качественно и в срок, Заказчик предоставляет доступы/информацию и своевременно принимает работы.
4. Порядок сдачи-приемки
Работы сдаются по Актам выполненных работ ежемесячно до 30 числа.
5. Стоимость и порядок оплаты
Ежемесячная оплата составляет 350 Евро. Оплата производится в эквиваленте в сомах по курсу Демир Кыргыз Интернешнл Банка в день конвертации после подписания Акта о приемке выполненных работ в течение 3 рабочих дней. Общая сумма выплат составляет за 3 месяца 1 050 Евро.

Приложение №1 к Договору № ___
Техническое задание на услуги по backend-разработке
•  Общие сведения
Проект: TilTap — AI-сервис многоязычного перевода для Telegram- и WhatsApp-ботов, ориентированный на журналистов, НКО и жителей приграничных регионов Ферганской долины (Кыргызстан, Таджикистан, Узбекистан).
Цель: разработка надёжной, масштабируемой и безопасной серверной архитектуры на Node.js/TypeScript, интеграция AI-сервисов перевода и обеспечение стабильной работы backend-части платформы TilTap.
•  Обязанности Исполнителя (по месяцам согласно плану проекта):
Июнь 2026: Настройка проекта и инфраструктуры backend (Node.js/TypeScript); проектирование схемы базы данных PostgreSQL; разработка базовых REST API endpoints; организация CI/CD pipeline.
Июль 2026: Реализация слоя интеграции AI-переводчиков; обеспечение безопасности API; оптимизация запросов к базе данных; начало документирования API (Swagger/OpenAPI).
Август 2026: Оптимизация производительности системы; масштабное тестирование backend, исправление ошибок (bug fixing); завершение документации (Swagger/OpenAPI); передача исходного кода и архитектурной документации Заказчику.

Надёжная и масштабируемая серверная архитектура на Node.js/TypeScript.
Настроенная база данных PostgreSQL с оптимизированными запросами.
Разработанные и задокументированные REST API endpoints (Swagger/OpenAPI).
Интегрированный слой подключения AI-сервисов перевода.
Обеспечение безопасности данных и защиты API.
Оптимизированная производительность backend-системы.
Передача исходного кода, документации по архитектуре и инструкций по развёртыванию Заказчику.
Согласование всех технических решений с Заказчиком и командой проекта.
